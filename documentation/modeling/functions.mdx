<Tip>
  Your number format in Francis determines whether you'll be using commas (US) or semicolons (EU) to separate function parameters. All examples below are presented in US format using commas.
</Tip>

<AccordionGroup>
  <Accordion title="avg(&#x22;row_name&#x22;[range_start:range_end])" defaultOpen={false}>
    The `avg()` function calculates the average of the values in the specified time series from `range_start` to `range_end`.

    ```typescript
    // Example
    avg("Revenue"[-3:0])
    ```
  </Accordion>

  <Accordion title="avg_ytd(&#x22;row_name&#x22;)" defaultOpen={false}>
    The `avg_ytd()` function returns the average of the values in a time series from the beginning of the fiscal year until and including the current month.&#x20;

    ```typescript
    // Example 
    avg_ytd("Revenue")
    ```
  </Accordion>

  <Accordion title="avg_last_year(&#x22;row_name&#x22;)" defaultOpen={false}>
    The `avg_last_year()` function returns the average of the values in the previous fiscal year. This function is influenced by your fiscal year settings.

    ```typescript
    // Example
    avg_last_year("Revenue")
    ```
  </Accordion>

  <Accordion title="avg_last(&#x22;row_name&#x22;,num_months)" defaultOpen={false}>
    The `avg_last()` function returns the average of the values in the time series going back *n* months, as defined by the `num_months` parameter.

    ```typescript
    // Example
    avg_last("Revenue",6)
    ```
  </Accordion>

  <Accordion title="median(&#x22;row_name&#x22;[range_start:range_end])" defaultOpen={false}>

    ```typescript
    // Example
    median("Revenue"[-3:0])
    ```
  </Accordion>

  <Accordion title="median_ytd(&#x22;row_name&#x22;)" defaultOpen={false}>

    ```
    // Example

    ```
  </Accordion>

  <Accordion title="median_last_year(&#x22;row_name&#x22;)" defaultOpen={false}>

    ```
  </Accordion>

  <Accordion title="median_last(&#x22;row_name&#x22;, num_months)" defaultOpen={false}>
    The `median_last()` function returns the median of the values in the time series going back *X* months, as defined by the `num_months` parameter.

    ```
  </Accordion>
