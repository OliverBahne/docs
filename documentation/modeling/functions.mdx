<Tip>
  Your number format in Francis defines whether you'll be using commas (US) or semicolon (EU) to separate function parameters. The examples below are all represented in US format using commas.
</Tip>

<AccordionGroup>
  <Accordion title="avg(&#x22;row_name&#x22;[range_start:range_end])" defaultOpen={false}>
    The `avg()` function calculates the average of the values in the specified time series from `range_start` to `range_end`.

    ```
    #Example
    =avg("Revenue"[-3:0])
    ```
